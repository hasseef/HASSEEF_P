# واجهة منصة حصيف – نسخة الهبوط الثنائية اللغة (Arabic/English Landing)

هذه الحزمة تحتوي على صفحة هبوط رئيسية لمنصة **حصيف** مع:

- واجهة عربية/إنجليزية (زر تبديل اللغة في الأعلى).
- تعريف عن المنصة.
- قسم الحسابات الثمانية مع روابط لصفحات فرعية لكل حساب.
- قسم المكوّنات الأساسية (Core Modules).
- قسم الربط مع رؤية 2030.
- قسم "التواصل والشراكات".
- شعارات رؤية 2030 + مبادرة تلبية + منصة حصيف في الهيدر والفوتر.

## هيكل المجلد

- `index.html` → صفحة الهبوط الرئيسية.
- `assets/css/style.css` → تنسيق الواجهة والهوية.
- `assets/js/app.js` → تمرير سلس + تبديل اللغة.
- `assets/img/` → ضع هنا الملفات:
  - `vision2030.png`
  - `talbiya.png`
  - `hasif.png`
- `accounts/` → صفحات فرعية لكل حساب (Emirate, Development, Government, Private, Nonprofit, University, Donor, Individual).

## طريقة الاستخدام

1. رفـع جميع الملفات إلى مستودع GitHub (مثلاً: `HASSEEF_P`).
2. وضع الشعارات الحقيقية داخل `assets/img/` بالأسماء نفسها.
3. تفعيل GitHub Pages من إعدادات المستودع (Source: `main` أو `master`, Folder: `/root`).
4. فتح الرابط الذي يولده GitHub Pages للتأكد من أن الصفحة تعمل بالعربية والإنجليزية.

هذا العمل يركّز على الهوية والعرض التعريفي، ويمكن لفريق التطوير لاحقًا ربطه بالـ Backend وواجهات الـ API.
