# منصة حصيف – واجهة الهوية (Landing Page)

هذا المجلد يحتوي على ملف **`index.html`** بهوية وتنسيق أولي لواجهة منصة حصيف،
جاهز للرفع على GitHub واستخدامه كنقطة بداية لواجهة المنصة (Landing / Frontend).

## الملفات الأساسية

- `index.html`  
  صفحة رئيسية RTL باللغة العربية توضّح:
  - تعريف المنصة
  - الحسابات الثمانية
  - المكوّنات الأساسية (Core Modules)
  - الربط مع رؤية 2030
  - قسم مبدئي للمطورين

- `assets/css/style.css`  
  تنسيق بسيط يعتمد ألوان الهوية:
  - الأخضر الأساسي: `#5AA044`
  - الذهبي: `#F8BE43`
  - الأزرق الداكن: `#0f172a`

- `assets/js/app.js`  
  سكربت بسيط للتمرير السلس بين أقسام الصفحة، ويمكن توسعته لاحقًا.

- `assets/img/`  
  ضع هنا شعارات:
  - `vision2030.png`  → شعار رؤية السعودية 2030
  - `talbiya.png`     → شعار مبادرة تلبية
  - `hasif.png`       → شعار منصة حصيف

## طريقة الاستخدام

1. ارفع هذا المجلد بالكامل إلى مستودع جديد في GitHub (مثلاً: `hasif-landing`).
2. أضف الشعارات الحقيقية داخل مجلد `assets/img/` بالأسماء المذكورة أعلاه.
3. فعّل GitHub Pages إن رغبت بعرض الصفحة مباشرة للزوار.
4. يمكن لفريق التطوير لاحقًا:
   - نقل هذه الواجهة إلى إطار عمل مثل Next.js أو React.
   - ربطها مع واجهات الـ API للخلفية (Backend) عند جاهزية الخدمات.

## ملاحظة

هذا العمل يركّز على **الهوية والنمط البصري** للمنصة فقط،
ولا يفرض أي هيكلية برمجية للخلفية أو الـ API. يمكن تعديل الكود بما يناسب
الهيكل النهائي للمشروع في GitHub.
